# QR-code-Generator
## Description
A QR-code Generator using JAVA Applets, Frames, and zxing library
### Usage
Just run the "Generator.java" file in src folder using command:
  ```bash
      $ javac Generator.java
      $ java Generator
  ```
Type in the text you want to encode.
